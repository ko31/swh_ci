# 将棋ウォーズ対局履歴取得ツール（CodeIgniter版）

## 概要

* 将棋ウォーズの履歴ページをスクレイピングして対局データを保存する PHP スクリプトです。
* [CodeIgniter](http://www.codeigniter.com/) ベースに作っています。
* スクレイピングには [Goutte](https://github.com/FriendsOfPHP/Goutte) を使っています。
